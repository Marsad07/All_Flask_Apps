# Flask_App_3

A mood-based Flask web app that lets users select their mood and view a themed response. Built for creative exploration and UI/UX practice.

##  Features

- Mood selection via dropdown
- Dynamic routing based on user input
- HTML templates with custom styling
- Modular Flask structure
